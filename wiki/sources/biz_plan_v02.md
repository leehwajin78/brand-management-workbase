# 브랜드 매니지먼트 사업기획서 v0.2

## 핵심 요약
5060 전문가의 경력을 브랜드·강의·제안 자산으로 번역해 실제 무대와 수익 기회로 연결하는 프리미엄 매니지먼트 서비스 전체 기획서.

## 주요 내용

### 사업 정의 3종
- **A:** 5060 전문가의 경력을 브랜드·강의·제안 자산으로 번역해 시장 무대로 연결하는 프리미엄 매니지먼트 서비스
- **B:** 직함 뒤에 숨어 있던 경험의 가치를 브랜드 언어로 바꾸는 8주 집중 서비스
- **C:** 정체성부터 강의안·제안서까지 통합 설계하는 브랜드 매니지먼트

### 해결 문제 3-Layer
| 층위 | 문제 | 브랜딩 함의 |
|------|------|-----------|
| Layer 1 | 경력 언어화 실패 | 자산 발굴·언어화가 진입 포인트 |
| Layer 2 | 자산 분산 | 마스터 브리프 기반 자산 파생 구조 |
| Layer 3 | 실행 진입 장벽 | PM형 일괄 대행 매니지먼트 |

### 8주 서비스 흐름 (5 Phase)
1. 자산 발굴·기획 (1~2주) → 마스터 기획 브리프 15P
2. 브랜드 구조화 (2~4주) → 브랜드 어휘집 + 메시지 구조
3. 에셋 제작 (3~5주) → [브랜드 에셋 12종](../concepts/brand_assets_12.md)
4. 지식 자산화 (5~7주) → PPT 원본 + B2B 제안서
5. 시장 연결 (7~8주) → 실행 가이드 + 1개월 매니지먼트

### 패키지 & 수익 구조
| 구분 | Option A | Option B |
|------|----------|----------|
| 가격 | 650만 원 | 880만 원 |
| 기간 | 5주 | 8주 |
| 외주 원가 | 350만 원 | 350만 원 |
| 순마진 | 300만(46%) | 530만(60%) |

### AI 개발 로드맵 (7개 항목)
1. ★★★ 사전 진단 툴 (2~3주)
2. ★★★ 마스터 브리프 자동 생성기 (3~4주)
3. ★★☆ 브랜드 언어 생성기 (1~2주)
4. ★★☆ 강의안 구조 자동화기 (2~3주)
5. ★★☆ B2B 제안서 자동화기 (2~3주)
6. ★☆☆ 고객 진행 대시보드 (3~4주)
7. ★☆☆ 진단→업셀 자동화 퍼널

### AI 대체 불가 영역
- 42문항 코칭 인터뷰 진행 자체
- 브리프 기반 최종 메시지 선택 및 편집
- 고객과의 신뢰 형성 및 정서적 안전감 제공
