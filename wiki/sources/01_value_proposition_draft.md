# Value Proposition 내부 기획 기준본 v2.0

> **문서 성격:** 내부 기획·운영 기준문서 (고객 공유 불가). 외부 공유 시 별도 버전 분기 원칙.

## 핵심 요약
이 문서는 두 파트로 구성됨:
1. **내부 기획 기준본 (§1~§7):** 타깃 정의, Pain 3-Layer, Gain 분석, 서비스 구조, 퍼널 설계, 가격 구조, 리스크 점검.
2. **AI 마스터 브리프 MVP 개발 목표 (§1~§12):** 운영 시간 압축, 리드 퍼널 구축, 바이브코딩 1인 개발 방향.

## 주요 인사이트

### 내부 기획 기준본
- **ROI 회수 논리 미검증 경고:** "자문 1건 + 특강 3건 = 880만 회수"는 파일럿 케이스 부재 상태. 외부 문서에 확정 수치 기재 금지.
- **퍼널 개선:** 기존 퍼널에서 진단→본계약 사이 신뢰 구축 단계(Step 3: 15분 무료 해석 통화 + 케이스 스터디 공유) 추가.
- **AI 역할 3단계 분리:** ① AI 초안 생성(30분) → ② 기획자 전략 검수(1~2시간) → ③ 디자이너 시각화(외주).
- **베타 파일럿 가격:** 350~440만(50% 할인), 케이스 스터디 활용 동의 조건.
- **실질 경쟁자 재정의:** 880만 원 결제 고려 고객이 비교하는 대안은 고가 퍼스널브랜딩 컨설팅(200~500만), 지인 소개 코치(100~300만), 현상 유지(0원).

### MVP 개발 목표
- **핵심 문제:** 건당 8~15시간 기획 리소스 소모 → 월 2건 상한 → 사업 확장 원천봉쇄.
- **3가지 핵심 가설:**
  - H1: 5060은 텍스트 입력보다 구술(녹음) 기반 인터뷰 툴에 더 강하게 반응.
  - H2: AI 초안의 기초 로직 퀄리티가 고객 납득 수준의 80%에 달할 것.
  - H3: AI 무료 진단 리포트 완독자의 최소 10%가 VVIP 상담 CTA 클릭.
- **가장 큰 리스크:** 프롬프트가 엉성할 경우 싸구려 문서(Hallucination) 발생 → 기술보다 **프롬프트 튜닝에 사활**.
