# 나다운 브랜딩 5060 코칭 가이드 42문항

- **원본 파일:** `raw/assets/나다운 브랜딩 5060 코칭 가이드 42문항.md`
- **핵심 요약:** 10~20년 이상 경력의 중장년 전문가를 코칭하기 위해 설계된 42개의 질문과 코칭 스크립트 매뉴얼. 이 비즈니스의 핵심 IP.

## 주요 내용
- **구성:** 5개 브랜딩 영역, 168개 패턴 코칭, 42개 프로필 연결.
- **4 파트 구조:**
  1. 과거 자원 탐색 (나는 어떤 삶을 살아왔는가)
  2. 강점·자산 발견 (나는 지금 무엇을 가지고 있는가)
  3. 미래 방향 설정 (나는 무엇을 원하는가)
  4. 브랜드 언어 구성 (나는 세상에 어떻게 말할 것인가)
- **특징:** 어설픈 공감이 아닌 경력자의 눈높이에 맞춘 '5060 특화 인사이트' 제공. 답변이 최종 아웃풋(원라이너, 가치 선언문, 페르소나 등)과 어떻게 연결되는지 명시.
